# setRemoteGUID

Assign a new GUID for this session at the remote VSCP host side.

## int setRemoteGUID(const uint8_t *pGUID);

### Parameters
* **pGUID** - 16-byte GUID to set.

### Returns
Return VSCP_ERROR_SUCCESS on success or an erro code in failure.
